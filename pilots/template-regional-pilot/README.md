# Regional Pilot Template

Use this folder as a starter scaffold for a new pilot.

## Replace Before Launch
- Pilot name and geography
- Co-stewards and governance model
- Scope, success metrics, and timeline
- Interoperability stack and consent policy

## Core Files
- `pilot-charter.md`
- `pilot-plan.md`
- `tooling-and-authority-map.md`
- `decision-log.md`
- `open-questions.md`
- `review-checklist.md`
